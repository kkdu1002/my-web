numpy

  1. 다차원의 배열 사용가능
  2. "ndarray"class사용
  3. 2차원 이상에서 axis사용
axis = 0 : 열
axis = 1 : 행
axis = -1 : 층
axis 0,1은 2차원이상에서 사용하며, -1은 3차원이상에서 사용함

행 : 가로 , 열 : 세로

배열 생성
변수명.Array([원소값,원소값]) > 1차원배열
변수명.Array([[원소값,원소값]],(원소값,원소값),[......]) > 2차원배열
[...]추가 할수록 x차원배열이 됨
변수명.ndim > 몇차원인지 알아내기
변수명.size > 항의 갯수
변수명.arange(갯수)
변수명.arange(시작,불포함 끝) , np.arange(시작,불포함끝,증감)
변수명.arange(시작,불포함 끝,dtype = data-type)
변수명.reshape(층수,행수,열수)
변수명.linespace(시작,포함 끝,갯수)
변수명.zeros((행수,열수))

배열 항목 지정 방법

  • 1차원 : 배열명[위치]
  • 2차원 : 배열명[행위치,열위치], 배열명[행위치][열위치]
  • 행위치에 "시작 : 불포함 끝" 사용가능
  • 행위치에 "시작 : 불포함 끝 : 증감" 사용가능
  • (리스트 뒤부터 시작) -값 사용가능
  • 위치지정을 배열로 사용가능
    • ex) 배열명[np.array([2,3,4])]
    내가 원하는 위치를 배열로 뽑을수있음
  • boolean 값으로 항목선택가능
    • 해당위치가 Ture면 값 선택가능
    배열명[np.array([ture,flase,ture])] 일때 ture랑 같은값 출력
    변수명[np.arange(x)] = 변수명 배열 방번호 0 ~ x-1까지 출력
